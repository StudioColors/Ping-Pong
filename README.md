# Пинг-Понг
В репозитории лежит прототип игры "Пинг-Понг". Это мультиплеерная игра, в рамках которой два игрока отбивают мячик из одного угла экрана в другой. Игрок, пропустивший мяч, проигрывает.

Игра реализована на языке Python с помощью библиотеки Pygame.

Игрок 1 управляет левой ракеткой с помощью клавиш WASD. Игрок 2 управляет правой ракеткой с помоью клавиш стрелок.
